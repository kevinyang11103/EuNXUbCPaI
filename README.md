# 智慧校园考试系统 python726

## 项目概述

《智慧校园考试系统》是一款基于 Python 开发的在线考试平台，旨在为校园提供便捷、高效的考试管理功能。

## 技术栈

- **后端**: Python 3, Django
- **前端**: HTML, CSS, JavaScript, Bootstrap
- **数据库**: SQLite

## 功能模块

- **用户管理**: 管理员账号管理
- **试题管理**: 创建、编辑、删除试题
- **考试管理**: 创建、发布、管理考试
- **成绩管理**: 查看和管理学生成绩

## 系统角色

- **管理员**: 负责系统的日常管理和维护

## 运行环境

- Python 3.x
- 网络环境：支持 HTTP 服务的网络环境

## 部署步骤

1. 确保Python 3.x环境已安装
2. 克隆或下载项目代码
3. 安装依赖包 `pip install -r requirements.txt`
4. 执行数据库迁移命令 `python manage.py makemigrations` 和 `python manage.py migrate`
5. 启动开发服务器 `python manage.py runserver`

- **灵活的用户权限管理**: 确保系统安全，易于管理
- **在线试题管理**: 简化出题过程，提高效率
